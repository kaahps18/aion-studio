# CREOVA AI - MVP

## Objetivo

Entregar uma versão utilizável o mais rápido possível.

## Funcionalidades da V1

### 1. Chat Inteligente
- Conversa
- Upload de imagens
- Upload de PDF

### 2. Prompt Studio
- Criar prompts
- Melhorar prompts
- Traduzir prompts
- Salvar prompts

### 3. Image Studio
- Gerar imagens
- Editar imagens
- Remover fundo
- Trocar fundo

### 4. Reverse Studio
- Receber link de vídeo
- Transcrever
- Resumir
- Gerar prompt baseado no vídeo

### 5. Projetos
- Criar projeto
- Salvar imagens
- Salvar vídeos
- Salvar prompts

### 6. Biblioteca
- Pesquisar arquivos
- Favoritos
- Histórico

### 7. Market Intelligence (Básico)
- Google Trends
- Calendário comercial
- Tendências por nicho

## Fora da V1

- Agentes automáticos
- Geração de vídeo própria
- Marketplace completo
- App mobile
