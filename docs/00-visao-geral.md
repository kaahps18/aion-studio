# AION Studio - Visão Geral

AION Studio é uma plataforma de IA para criação de conteúdo, e-commerce e marketing.

O objetivo é permitir que uma pessoa crie campanhas, prompts, imagens, vídeos, análises de tendências e materiais de venda em um único lugar.

## Missão

Criar uma plataforma simples, acessível e modular para ajudar criadores, lojistas e pequenos negócios a vender melhor usando inteligência artificial.

## Princípios

1. Simples de usar.
2. Gratuita no início.
3. Leve para computadores fracos.
4. Modular.
5. Focada em produtividade.
6. Preparada para chat, imagem, vídeo, prompts e inteligência de mercado.

## Módulos principais

- Chat Studio
- Prompt Studio
- Image Studio
- Video Studio
- Reverse Studio
- Market Intelligence
- Product Studio
- Biblioteca
- Projetos
